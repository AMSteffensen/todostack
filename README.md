# Fullstack TODO CRUD APP 
using Django Rest Framework and React frontend

Features:
- Create new user
- Login/logout
- Add todo
- Edit todo
- Delete todo

# Run dev local

Backend deployed to Pythonanyw
Frontend deployed to Netlify 

Run Local backend
- `cd backend`
- `pip3 install -r requirements.txt`
- `python3 manage.py runserver`

Run frontend Local
- `cd frontend`
- `npm install`
- `npm run start`

Deploy frontend
- `cd frontend`
- `npm run build`
- `netlify deploy --prod`
